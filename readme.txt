== Description ==

Hopos Slider Plugin is highly customizable and flexible. Is responsive and can adjust to any resolution. Can be used any post type and any taxonomy. In each post can be configured options such as video, large image, link, lightbox or target page.

See some examples of the plugin in operation: <a href="http://srvalle.com/plugins/hopos/" target="_blank">Hopos Slider Lite</a>

<h3>Main Features</h3>

* Any post type and any taxonomy;<br>
* Responsive;<br>
* Crossbrowser compatibility;<br>
* Desktop, tablet and mobile support;<br>
* Flexible and unlimited customization of styles;<br>
* Touch Swipe (mobile devices);<br>
* Drag enable;<br>
* CSS3 animations / jQuery fallback;<br>
* Video (Youtube, Vimeo, MP4 and FLV), Photo and Audio;<br>
* Lightbox or target page;<br>
* Navigation, play/pause, previous/next;<br>
* Ajax shortcode settings (admin);<br>
* Thumbnails in any size;<br>
* Excerpt with the number of words;<br>
* Order By, Order;<br>
* Pagination style (bullets or numbers);<br>
* Auto Slide;<br>
* Delay Auto Slide;<br>

== Installation ==

**INSTALLATION:** The easiest way to install Hopos Slider Lite is to login to you WordPress dashboard, go to Plugins >> Add New, search for Hopos Slider Lite, and click to install. You can also download the zip file from this page and upload it from the Plugins >> Add New > Upload page.

**HOW TO USE:** Just go to the menu plugin called Hopos Lite, and choose "Add New". Set all options on edit page and save. Then go back to the home page of the plugin and copy the shortcode. After paste the shortcode within a post, page or widget(Text). You can also use the plugin directly in the page code, using do_shortcode("[hopos id='x']");
